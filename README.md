# HTO Skills

Skills de **High Ticket Operator** para Claude. Sirven tanto en **Claude Code**
(como plugins, vía este marketplace) como en **Claude.ai / Cowork** (subiendo la
skill como ZIP).

## Skills disponibles

### `oferta-irresistible`
Facilitador estratégico que construye o audita ofertas high-ticket para el
mercado hispanohablante. Dos modos:
- **CREAR** — te guía bloque a bloque (inventario → ICP → promesa → mecanismo →
  stack → bonuses → garantía → pricing → escasez → stress test) y genera un
  documento interno de negocio + una carta de presentación para tu lead.
- **REVISAR** — auditas una oferta existente contra el framework y recibes un
  informe de huecos y mejoras priorizadas.

---

## Instalar en Claude Code

```
/plugin marketplace add nicolascerezoo/hto-skills
/plugin install oferta-irresistible@hto-skills
```

Para actualizar más adelante:

```
/plugin marketplace update hto-skills
```

---

## Usar en Claude.ai / Cowork

1. Descarga el ZIP de la skill: `dist/oferta-irresistible.zip`.
2. En Claude.ai ve a **Ajustes → Capabilities → Skills**.
3. Sube el ZIP. La skill queda disponible en tus chats.

> El ZIP no se auto-actualiza. Cuando salga una versión nueva, vuelve a
> descargarlo y súbelo de nuevo.
